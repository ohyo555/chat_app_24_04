### polling vs SSE(Server-Sent-Events)
- polling: 클라이언트가 n초 간격으로 request를 서버로 날려서 response를 전달받는 방식
- SSE: js를 이용해서 서버의 데이터를 실시간, 지속적으로 Streaming하는 기술 (server → client 단방향)
